# EcommerceStore 
> Built a full stack E-commerce web store using .Net and React

## Key features
1. fetch products from database
2. view products
3. add products to cart
4. pagination
5. filtering /sorting/ searching product
6. theme light/dark

## Technologies learnt
1. Entity framework
2. REST api
3. typescript
4. Material UI
5. React
6. Axios
7. React Context
8. Redux
9. Heroku hosting
