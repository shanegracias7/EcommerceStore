# EcommerceStore 
> Built a full stack E-commerce web store using .Net and React

## Key features
1. view products
2. add products to cart
3. pagination
4. filtering /sorting/ searching product
5. theme light/dark

## Technologies learnt
1. Entity framework
2. REST api
3. typescript
4. Material UI
5. React
6. Axios
7. React Context
8. Redux
9. Heroku hosting
